Complete System Usage Guide

1 Dashboard Overview

The Dashboard shows real-time statistics including total investors, active batches, pending requests, total investments, total profits, pending withdrawals, insurance pool balance, and total referrals.

📊 Example: Reading Dashboard Stats

Situation: You want to know how many investors have joined this month.

Action: Look at the "Total Investors" card - it shows the cumulative count of all registered investors.

Result: If it shows 150, you know 150 investors are registered in the system.

💡 Tip: Click on any stat card to directly navigate to that section for detailed management.

2 Batch Management

Create and manage investment batches with customizable parameters including amount, profit percentage, duration, and total slots.

📦 Example 1: Creating a New Batch

Goal: Create a Summer 2025 batch with Rs.50,000 investment, 50% profit, 90 days duration, and 100 slots.

Steps: Navigate to Batches → Fill form (Batch Name: Summer 2025, Amount: 50000, Profit%: 50, Duration: 90, Slots: 100) → Click "Create Investment Batch".

✅ Result: New batch appears in the batches table with status "Open".

🔄 Example 2: Batch Lifecycle Management

Goal: Update batch status from "Open" → "Running" → "Sold" → "Completed".

Steps: Find batch → Click "Edit" → Change status field → Save.

✅ Result: Batch status updates and progress bar reflects current stage.

⏰ Example 3: Setting Auto-Expiry

Goal: Batch auto-expires after 30 days if not filled.

Steps: During batch creation, set "Auto-Expiry (Days)" to 30 → Create batch.

✅ Result: If batch not filled in 30 days, system auto-cancels and refunds investors.

3 Investor Management

Add, edit, and manage investor profiles with permission controls for chat, voice, and video features.

👤 Example 1: Adding a New Investor

Goal: Register investor "Ahmed Khan" with email ahmed@example.com.

Steps: Investors section → Fill name and email → Click "Add Investor".

✅ Result: Investor appears in investors list with initial balance Rs.0.

🔒 Example 2: Toggle Investor Chat Permission

Goal: Disable chat for a specific investor due to policy violation.

Steps: Find investor in list → Click "Toggle Chat" button → Confirm.

✅ Result: Investor's chat permission changes from ON to OFF (red badge).

📥 Example 3: Export Investor Data (Backup)

Goal: Download complete investor list as CSV for backup.

Steps: Investors section → Click "Export Investors CSV" button.

✅ Result: CSV file downloads with all investor data (Name, Email, Phone, Investment, Profit, Balance).

4 Investment Request Processing

Review and approve/reject investment requests with mandatory payment screenshot verification.

💰 Example 1: Approving an Investment Request

Goal: Investor invested Rs.100,000 and uploaded payment proof.

Steps: Investment Requests section → Click "Approve" on request → Verify screenshot → Confirm approval.

✅ Result: Investment status changes to "Active", investor balance increases by Rs.100,000.

❌ Example 2: Rejecting Invalid Request

Goal: Investor submitted request without payment screenshot.

Steps: Investment Requests section → Click "Reject" → Provide reason → Confirm.

✅ Result: Request status changes to "Rejected", investor notified.

5 Profit Distribution

Distribute profits to investors based on batch performance (50% profit sharing model).

📈 Example 1: Manual Profit Distribution

Goal: Credit Rs.25,000 profit to investor "Sarah Ali".

Steps: Profits section → Select investor → Enter amount 25000 → Select month → Click "Distribute Profit".

✅ Result: Investor profit increases by Rs.25,000, available balance increases similarly.

🏭 Example 2: Batch Profit Distribution

Goal: Batch completed with Rs.200,000 net profit, distribute 50% to investors.

Steps: Batches section → Find batch → Click "Distribute Profit" → Review distribution → Confirm.

✅ Result: System automatically distributes profits proportionally to all investors in batch.

🎁 Example 3: Bonus Profit for Long-term Investors

Goal: Give bonus profit to investors who stayed for 1+ year.

Steps: Profits section → Select investor → Enter bonus amount → Select "Profit Type: Bonus".

✅ Result: Bonus profit credited separately, visible in profit history with "bonus" type.

6 Withdrawal Management

Process withdrawal requests with mandatory approval screenshot for audit trail.

💸 Example 1: Approving Withdrawal

Goal: Investor requests Rs.50,000 withdrawal.

Steps: Withdrawals section → Click "Approve" → Upload transaction screenshot → Confirm.

✅ Result: Withdrawal approved, investor balance decreases by Rs.50,000.

7 Insurance Pool & Loss Protection

Manage insurance pool to protect investors from losses (50% recovery coverage).

🛡️ Example 1: Adding Funds to Insurance Pool

Goal: Add Rs.500,000 to insurance pool.

Steps: Insurance section → Enter "500000" in Add Funds → Click "Add to Pool".

✅ Result: Insurance pool balance increases by Rs.500,000.

🔄 Example 2: Processing Loss Claim

Goal: Investor suffered Rs.100,000 loss, claim 50% from insurance.

Steps: Insurance section → Click "Process Loss Claim" → Select investor → Enter loss amount 100000 → Confirm.

✅ Result: Investor receives Rs.50,000 recovery, insurance pool decreases by Rs.50,000.

8 Referral & Affiliate System

Multi-level referral tracking with automatic commission calculation.

🔗 Example 1: Setting Referral Commissions

Goal: Set Level 1=10%, Level 2=5%, Level 3=2.5% commissions.

Steps: Referrals section → Enter percentages → Click "Save Settings".

✅ Result: New referrals will earn commissions at these rates.

🏆 Example 2: Referral Milestone Bonus

Goal: Investor reached 10 referrals, give Rs.15,000 bonus.

Steps: Referrals section → Set milestone values → System auto-detects and awards bonus.

✅ Result: Investor receives bonus profit credited to their account.

9 Live Chat Support

Real-time chat with investors and buyers for support and communication.

💬 Example 1: Chatting with Investor

Goal: Respond to investor question about batch status.

Steps: Live Chat section → Select investor from left panel → Type message → Press Enter or click send.

✅ Result: Message delivered instantly, investor receives notification.

🔇 Example 2: Disabling Chat for Specific User

Goal: Temporarily disable chat for a problematic user.

Steps: Investors section → Find user → Click "Toggle Chat" button.

✅ Result: User's chat permission disabled, they cannot send messages.

10 Data Backup (CSV Export)

Export all critical data for backup and disaster recovery.

💾 Example 1: Backup All Investors

Goal: Download complete investor database as CSV.

Steps: Investors section → Click "Export Investors CSV" → Save file.

✅ Result: CSV file contains all investor records for safe keeping.

📋 Example 2: Backup All Buyers

Goal: Download complete buyer database as CSV.

Steps: Buyers section → Click "Export Buyers CSV" → Save file.

✅ Result: CSV file contains all buyer records for safe keeping.

11 Leaderboard & Gamification

Track top performers and award badges for achievements.

🏅 Example 1: View Top Investors

Goal: See who invested the most this month.

Steps: Leaderboard section → Select "Top Investors" from dropdown.

✅ Result: List shows investors ranked by total investment with badges.

🎖️ Example 2: Auto-Assign Badges

Goal: Assign badges to eligible investors.

Steps: Gamification section → Click "Auto-Assign Badges".

✅ Result: System assigns badges (Top Investor, Early Investor, High Earner) automatically.

12 Security & Anti-Fraud

Protect system from fraudulent activities with IP blocking and monitoring.

🚫 Example 1: Block Suspicious IP

Goal: Block IP 192.168.1.100 from accessing system.

Steps: Security section → Enter "192.168.1.100" → Click "Block IP".

✅ Result: IP address blocked, cannot access investor/admin panels.

13 Automation System

Automate repetitive tasks for efficient management.

🤖 Example 1: Enable Auto Profit Distribution

Goal: Automatically distribute profits when batches complete.

Steps: Automation section → Toggle "Auto Profit Distribution" ON → Save Settings.

✅ Result: System automatically distributes profits when batch status changes to "Profit Distributed".

14 Sub-Admin Management

Delegate tasks to team members with role-based permissions.

👥 Example 1: Add Finance Sub-Admin

Goal: Add finance@sohrab.com as Finance sub-admin.

Steps: Sub-Admins section → Enter email → Select role "Finance" → Click "Add Sub-Admin".

✅ Result: Sub-admin added with finance-related permissions only.

15 Marketing Campaigns

Create promotional campaigns to attract more investors.

📢 Example 1: Create Discount Campaign

Goal: Offer 15% discount for first-time investors.

Steps: Marketing section → Enter campaign name → Set discount 15% → Click "Create Campaign".

✅ Result: Campaign active, investors see discount on investment page.

Emergency Recovery Instructions

If system crashes or data is lost:

  1. Use exported CSV files (Investors and Buyers) to restore user data
  2. Contact Supabase support to restore database from backups
  3. All transaction records are stored in Supabase and can be recovered
  4. Regular CSV exports recommended (weekly or monthly)
